搜索

GET: /search

对于红旅动漫页面: http://www.hltm.tv/search/?k=海贼王

  • page Number 页数

示例: /search

动漫详情

GET: /detail

对于红旅动漫页面: http://www.hltm.tv/view/972.html

  • url String 对于红旅网站动漫详情页面的url(需要Encoder)

示例: /detail

连载动漫/完结动漫

GET: /list

对于红旅动漫页面: http://www.hltm.tv/list/7.html

  • ver Number 版本, 全部:"" TV番组:1 OVA:2 OAD:3 剧场版:4 特别版:5 真人版:6 动画版:7 其他:100
  • letter String 字母 全部:"" A:A B:B...以此类推 0-9:100
  • sort Number 排序方式 最新更新:"" 热门播放:1
  • tab Number 类型 连载动漫:7 完结动漫:18
  • page Number 页数

示例: /list
