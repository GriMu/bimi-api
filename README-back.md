## 如何开始

- 打开终端（windows下的命令行窗口）
- git clone https://github.com/tomoya92/hltm-api.git
- cd hltm-api
- npm install 
- npm start
- 打开浏览器输入 http://localhost:3000 回车

本人也提供了一份线上访问地址 https://hltm-api.tomoya.cn
